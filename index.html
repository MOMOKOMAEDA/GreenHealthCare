みんなの1日

  • 7:00
    起床 - さわやかな一日の始まり +

    利用者のペースに合わせた起床時間です。毎朝、スタッフが起床のサポートを行い、快適な一日を迎えられるようお手伝いします。また、個別ニーズに応じて時間の調整も可能で、日々の体調やリズムを尊重しながら支援します。

  • 8:00
    朝食 - 栄養豊かな朝食で元気に +

    栄養バランスを考えた朝食を提供し、利用者の食事状況や嗜好に応じて柔軟に対応しています。
    特別な食事制限やアレルギーがある場合にも配慮し、季節感あふれるメニューで楽しみながら食事できるよう工夫しています。

  • 9:00
    健康チェック - 健康管理で安心の毎日 +

    毎日の健康状態を確認するため、血圧や体温のチェック、服薬確認を行います。
    専任のスタッフが対応し、緊急時には医療機関と連携できる体制を整えているため、安心して生活していただけます。

  • 10:00
    リハビリ・趣味活動 - 身体と心の活力アップ +

    身体機能の維持や認知機能向上のため、リハビリテーションや趣味活動を取り入れています。個別プランに基づき、柔軟にプログラムを調整しながら、利用者が楽しく参加できるようサポートしています。

  • 12:00
    昼食 - バランスの取れた昼食 +

    昼食は、旬の食材を取り入れた栄養豊富なメニューを用意しています。利用者の食べやすさに配慮し、味覚や噛む力に合わせた調整も行います。食事は楽しみのひとつであるため、提供方法にも工夫を凝らしています。

  • 13:00
    アクティビティ
    自由時間 - 楽しみとリフレッシュの時間 +

    アクティビティや自由時間では、趣味活動やグループでの活動、静かな読書など、自由に過ごせる時間です。利用者の好みに合わせたプログラムを取り入れ、リラックスして楽しめる空間を提供しています。

  • 15:00
    おやつタイム - くつろぎの午後のひととき +

    午後のおやつの時間は、気軽にリラックスできるひとときです。軽いおやつやお茶を楽しみながら、利用者同士のコミュニケーションも深められる場となっています。特別なおやつも提供し、楽しみを増やしています。

  • 17:00
    夕食 - 一日の終わりを彩る夕食 +

    夕食は、一日の終わりを締めくくる大切な時間です。栄養バランスに配慮したメニューを提供し、ゆったりとした雰囲気で食事を楽しんでいただけるよう工夫しています。利用者の嗜好や健康状態に合わせた対応も可能です。

  • 18:00
    入浴タイム - 心身を癒す入浴タイム +

    リラックスできる入浴タイムは、個別の介助が必要な方にも対応し、快適なバスタイムをサポートしています。プライバシーを重視しながら、利用者がゆっくりと疲れを癒せる時間を確保しています。

  • 19:00
    休憩時間 - くつろぎの時間で一息 +

    一日の終わりに向けて、ゆったりと休む時間です。居室やリビングでのんびりとしたひとときを過ごしながら、明日に備えて心身をリフレッシュします。利用者のリズムに合わせて自由に過ごせます。

  • 21:00
    就寝 - 穏やかな夜の休息を +

    安心して就寝できるよう、居室環境や室温の調整、リラックスできる空間を提供しています。スタッフが見守り、夜間の急な対応も可能な体制を整えています。利用者が安心して安らかな夜を迎えられるようサポートしています。

入居までの６ステップ

見学・資料請求
01 >>
自分好みのあなたに合ったニーズの施設を見つけましょう。
入居申込
02 >>
入居申込書、かかりつけ医による診療情報提供書、健康保険証、介護保険証の写しなどを提出します。
施設にて面談
03 >>
施設の生活相談員等による面談。
健康状態や経済状況、家族情報などの確認を行われます。
仮入居
04 >>
実際の生活環境を短期間体験することで、入居後の生活を具体的に確認することができます。
契約・入居準備
05 >>
契約手続きから入居準備、必要な書類や生活用品の確認まで、スタッフがサポートします。
入居
06 >>
入居当日は、スタッフが施設案内や生活サポートを行い、安心して新生活を始められます。
